Mini-Toolbar        


               Help                                        (F1)

Help Contents (Context-sensitive).

               New                                        (CTRL + N)

Create a new .app or .ppg file.

               Open                                        (CTRL + O)

Open an existing .app or .ppg file for editing.

               Save                                        (CTRL + S)

Save the current .app or .ppg file.

Îž Note: This will also convert strings to variables and convert between BuildTypes. For example, "C:\Program Files" becomes %ProgramFiles% and changes an ssApp to a ppApp (or vise-versa) including all the auxiliary support files and folders. See Automatic Variable Substitution Chart for details.

               Explore Working Folder                (F2)

Opens Explorer to the current working folder listed in the status bar.

               View Variables List                (F3)

See Automatic Variable Substitution Chart for details.

               View Source in Notepad                (F4)

Take a look at the .app/.ppg file at its most basic level.

               Extract Icons                        (F10)

Useful for capturing some folder icons for SetupS to use.

               Screen Capture                        (F11)

Useful for capturing some screen-shots for ssWPI.

               Grab Clipboard Picture                (F12)

An alternative way to obtain some screen-shots for ssWPI.

               SetupS Test                                (F8)

Handy debugging tool for testing this package through SetupS. Temporarily turns on Fadertainer to see how the Fader graphic will look.

               Build!                                        (F5)

Builds the deployment package. See Builds Menu for details.
