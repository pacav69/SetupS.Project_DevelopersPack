md5sum

md5sum        v.1.3.0
Copyright Â© 2008-2009 Kai Liu.  All rights reserved.

http://code.kliu.org/misc/hashutils/

A simple command-line check-summing/hashing utility for Windows, similar to the ones found on most Unix-like systems. The md5sum utility found in this package will produce output similar to and compatible with that of its Unix counterpart.

The -c switch is not supported; to verify checksum files, use HashCheck. All of these utilities support recursive directory traversals with the -r switch.
